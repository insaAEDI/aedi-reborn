Plusieurs services sont hébergés sur le serveur : le site web, shareif, etc. Voici un résumé de comment sont matérialisés ces services sur le serveur.

===== HTTP =====

Il y a deux serveurs HTTP : nginx et httpd. Httpd sert à faire tourner SVN, nginx s'occupe de tout le reste. Ils sont controllables avec init.d.

===== Shareif =====

Shareif est un site web placé dans /data/www/shareif. À ce site s'additionnent deux programmes : gitweb et svn, tous deux controllables avec init.d.
Les projets des gens sont stockés avec mySQL, dans la base shareif2. Cette base est accessible par root ou par l'utilisateur shareif/htpud24c. Shareif est fait de telle sorte que tout étudiant de l'INSA peut y créer un dépôt. Lorsqu'un utilisateur inconnu se connecte au service, il est automatiquement ajouté dans la base de données. C'est pour cela qu'on ne peut pas ajouter un utilisateur à un dépôt existant avant que ce dernier ne se soit connecté au moins une fois.

===== Ifaedi =====

Il s'agit d'un site web placé dans /data/www/ifaedi.
